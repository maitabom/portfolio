# Currículos / Resumes

Coloque seus arquivos PDF de currículo nesta pasta com os seguintes nomes:

- `curriculo-pt.pdf` - Currículo em Português
- `resume-en.pdf` - Resume in English
- `curriculum-es.pdf` - Currículum en Español

## Instruções

1. Prepare seus currículos em PDF nos três idiomas
2. Renomeie os arquivos conforme os nomes acima
3. Coloque-os nesta pasta (`public/resumes/`)
4. Os botões na página "Sobre" farão o download automático dos arquivos

## Observação

Se você não tiver currículos em todos os idiomas, pode:
- Criar apenas os que você tem disponíveis
- Modificar o código em `src/app/components/about/about.component.ts` para remover os idiomas não utilizados
- Ou manter todos os botões e criar os PDFs posteriormente
